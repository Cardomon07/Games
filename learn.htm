HTML Tags Showcase

1. <a> Tag

Definition: The <a> tag defines a hyperlink, which is used to link from one page to another.
Usage: Use the <a> tag to create links to other pages, anchors within the same page, email addresses, or any URL. 
<a href="#">This is a link</a>
This is a link

2. <h1> Tag

Definition: The <h1> tag is used to define the most important heading in HTML, with <h1> being the highest level.
Usage: Use the <h1> tag for main titles or headings to give them more weight in SEO. 
<h1>This is a Heading 1</h1>

This is a Heading 1

3. <p> Tag

Definition: The <p> tag defines a paragraph.
Usage: Use the <p> tag to add text content in paragraph form. 
<p>This is a paragraph.</p>

This is a paragraph.

4. <img> Tag

Definition: The <img> tag is used to embed images in HTML.
Usage: Use the <img> tag to display images on the webpage by specifying the source URL in the src attribute. 
<img src="https://via.placeholder.com
/150" alt="Example Image">

5. <ul> and <li> Tags

Definition: The <ul> tag defines an unordered list, and the <li> tag defines a list item.
Usage: Use the <ul> and <li> tags to create bulleted lists. 

<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
  • List item 1
  • List item 2
  • List item 3

6. <ol> and <li> Tags

Definition: The <ol> tag defines an ordered list, and the <li> tag defines a list item.
Usage: Use the <ol> and <li> tags to create numbered lists. 

<ol>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ol>
  1. List item 1
  2. List item 2
  3. List item 3

7. <table>, <tr>, <th>, and <td> Tags

Definition: The <table> tag defines a table, <tr> defines a row, <th> defines a header cell, and <td> defines a data cell.
Usage: Use these tags to create and structure tables. 

<table border="1">
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>
Header 1 Header 2
Data 1 Data 2

8. <form>, <input>, and <button> Tags

Definition: The <form> tag defines an HTML form for user input, <input> is used for input fields, and <button> defines a clickable button.
Usage: Use the <form> tag to collect user input and <input> for different types of fields. Use <button> for form submission or other actions. 

<form>
    <input type="text" placeholder=
    "Enter text"> <button type=
     "submit">Submit</button>
</form>

Definition: The <div> tag is a block-level container element, while <span> is an inline container element.
Usage: Use <div> for grouping and styling block-level content, and <span> for styling inline content. 

<div>This is a div</div>
<span>This is a span</span>

11. <mark> and <small> Tags

Definition: The <mark> tag highlights text, while the <small> tag makes text smaller.
Usage: Use <mark> to highlight text and <small> to reduce the font size for less important text. 

<mark>This text is highlighted.</mark>
<small>This text is small.</small>

12. <blockquote> Tag

Definition: The <blockquote> tag is used to define a block of text that is a quotation from another source.
Usage: Use <blockquote> for longer quotations that need to be highlighted.

13. <code> Tag

Definition: The <code> tag is used to display a single line of code.
Usage: Use <code> for inline code snippets within a text. 
<code>This is code text.</code>
This is code text.

14. <pre> Tag

Definition: The <pre> tag is used to display preformatted text, preserving spaces and line breaks.
Usage: Use <pre> for text that requires exact formatting, such as code blocks.

Definition: The <br> tag inserts a line break, and the <hr> tag inserts a horizontal rule.
Usage: Use <br> for line breaks within text, and <hr> for separating sections.

16. <abbr> and <cite> Tags

Definition: The <abbr> tag is used for abbreviations or acronyms, and the <cite> tag is used to reference the title of a creative work.
Usage: Use <abbr> to provide an expanded form of abbreviations and <cite> to reference titles of books, articles, etc.

18. <details> and <summary> Tags

Definition: The <details> tag specifies additional details that the user can view or hide, while the <summary> tag provides a heading for the <details> content.
Usage: Use <details> and <summary> to create expandable content that users can click to reveal or hide.
